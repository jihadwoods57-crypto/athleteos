## Decisions (with rationale)
- Fable 5 never merges to master; the founder integrates. Rationale: LLM output is reviewed before it ships.
- 2026-07-11 ‚Äî Trainer identity is server-hydrated into RT (never `RT.ob` scratch, never hardcoded personas like "Tracy Boone"/"Coach Mark"). Rationale: profile must survive reinstall/new device and never show another persona. SETTLED ‚Äî do not re-litigate.
- 2026-07-11 ‚Äî Tab-root role dashboards get chevron-less headers (`titleHead`), not `backHead(...,'profile')`. Rationale: back on a root routed cross-role into the athlete profile. SETTLED.
- 2026-07-11 ‚Äî QR generation is a from-scratch, dependency-free encoder (no CDN in the WebView, no npm dep). SETTLED.
- 2026-07-11 ‚Äî Practice HQ visual lane: trainer purple as accent inside the existing dark redesign system; Athlete Blue remains the athlete spine. SETTLED (design taste, founder may override).
- 2026-07-11 ‚Äî Streak urgency is grace-calibrated, never flat: strong loss-aversion copy ("ends tonight") ONLY when grace is genuinely spent (`graceUsedRecently`); mild "covered" framing otherwise. Honesty over engagement pressure. SETTLED.
- 2026-07-11 ‚Äî Home ribbons/prompts render as siblings of `strip()`, never children ‚Äî strip owns `data-go="score-breakdown"` and nested data-go targets must not compete. SETTLED (pattern for future Home surfaces).
- 2026-07-12 ‚Äî Router wires [data-go]/[data-act] ONCE at render; mount()-injected elements MUST self-wire clicks (bug found 3√ó). SETTLED.
- 2026-07-12 ‚Äî supabase-js resolves network failures into `{error}` WITHOUT throwing ‚Äî offline detection must use the `{error:true}` sentinel from the fetch layer, never a catch around the loader. SETTLED (fetchMyTeams/fetchMyPractices now follow fetchMyPracticeIdentity).
- 2026-07-12 ‚Äî Never render a control that persists nothing (data-toggle-group theater): make it honestly inert + labeled preview, or remove it. SETTLED (checkin, coachVoice, quiet-hours).
- 2026-07-12 ‚Äî CSS scoping traps: `.wb2` requires `.water-btns` ancestor, `.chp` requires `.chip-row` ‚Äî bare use silently unstyles. Check the selector, not just the class name.
- 2026-07-12 ‚Äî Gallery meal photos are flagged `live:false` and disclosed with the amber NON-LIVE badge; scoring stays unchanged until the founder picks an enforcement rule (Option A block-scoring recommended) and authorizes any detection backend. Rationale: honest disclosure ships free; enforcement/detection is a product-integrity + migration/model decision. Spec: `reports/2026-07-12-t2-integrity-spec.md`. FOUNDER-GATED.
- 2026-07-12 ‚Äî Notifications stays a routed full screen, never a drawer/panel: rows are actionable accountability moments, `#notifications` already participates in back-nav/deep-linking, and a drawer is a per-surface fork. (Design recommendation in T5; becomes SETTLED if the founder approves the build.)
- 2026-07-12 ‚Äî Security migrations are authored on-branch, never applied to live (`0053_authz_hardening.sql` awaits founder `test:rls` + apply at go-live). Reaffirms the author-only guardrail.
