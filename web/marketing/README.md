al., Br J Sports Med 2018 meta-analysis, 49 trials, n=1,863); ~2× weight loss for

REDs (IOC consensus, Br J Sports Med 2023). Card 07's $20.5M figure is the House
settlement's year-one per-school revenue-sharing cap (2025–26). All product claims
